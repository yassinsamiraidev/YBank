# YBank 🏦

A secure, lightweight banking system built entirely in Python.
YBank gives you full control over your finances from the command line.

---

## Features
- 🔐 Password protected accounts
- 💳 Unique 12-digit account ID generation
- 💰 Deposit and withdraw funds
- 🔄 Transfer money between accounts
- ✅ Activate and deactivate accounts
- 🗑️ Delete accounts
- 💵 Real-time balance checking
- 💾 Persistent JSON storage

---

## Installation

Make sure Python is installed, then run:
```bash
pip install pyinputplus
```

---

## How To Run
```bash
python Chatbot.py
```

---

## How To Use
```
1 - Create Account    → Register a new bank account
2 - Delete Account    → Permanently remove account
3 - Add Money         → Deposit funds
4 - Withdraw Money    → Withdraw funds
5 - Know Balance      → Check current balance
6 - Activate Account  → Re-enable a deactivated account
7 - Deactivate        → Temporarily disable account
8 - Transfer Money    → Send money to another account
9 - Quit              → Exit the program
```
